
### 3.1 Avarana Shakti (à¤†à¤µà¤°à¤£ à¤¶à¤•à¥à¤¤à¤¿) â€” Veiling Power

**Function:** Hides the source (Brahman) from the observer


**Analogy:** Like fog hiding the sun

### 3.2 Vikshepa Shakti (à¤µà¤¿à¤•à¥à¤·à¥‡à¤ª à¤¶à¤•à¥à¤¤à¤¿) â€” Projecting Power

