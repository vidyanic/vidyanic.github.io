## 3.5 LIGHT AS SYNCHRONIZER

### Surya (Sun) Entrainment

```
LIGHT-CLOCK CONNECTION:
=======================

SUNRISE:
Light -> Eyes -> SCN -> "Wake up" signals
Cortisol rises, melatonin falls

SUNSET:
Darkness -> No light input
Melatonin rises, cortisol falls

SURYA = PRIMARY TIMEKEEPER
Body follows sun = Body follows Dharma
```
