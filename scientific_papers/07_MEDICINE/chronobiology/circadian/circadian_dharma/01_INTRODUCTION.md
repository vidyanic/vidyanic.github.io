## 1.2 WHAT ARE CIRCADIAN RHYTHMS?

**Circadian (Latin):**
- Circa = About
- Dies = Day
- ~24-hour biological cycles

| Feature | Description |
|---------|-------------|
| **Endogenous** | Internally generated |
| **Entrained** | Synchronized by light/dark |
