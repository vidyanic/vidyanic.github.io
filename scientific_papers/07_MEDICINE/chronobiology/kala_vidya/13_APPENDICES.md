## Appendix C: Brahma Muhurta Research

### C.1 Time: 1.5 hours before sunrise

**Neurophysiology:**
- End of REM cycle
- Melatonin still present
- Cortisol beginning rise
- Brain in alpha/theta

**Traditional claims:**
- Mind receptive
- Sattva predominant
- Best for meditation
- Spiritual practice optimal

**Modern support:**
- Transition state = flexibility
- Quiet environment
- Before stress hormones peak

## Appendix D: Clock Genes Summary

### D.1 Core Clock Genes

| Gene | Function |
|------|----------|
| CLOCK | Transcription activator |
| BMAL1 | Partner of CLOCK |
| PER1, PER2, PER3 | Period (negative feedback) |
| CRY1, CRY2 | Cryptochrome (negative feedback) |
| REV-ERBα/β | Additional regulation |

### D.2 Mechanism

```
Morning: CLOCK + BMAL1 activate PER/CRY
Day: PER/CRY proteins accumulate
Night: PER/CRY inhibit CLOCK/BMAL1
Late night: PER/CRY degrade
Morning: Cycle restarts

Period: ~24 hours
Entrainment: Light via retina
```
