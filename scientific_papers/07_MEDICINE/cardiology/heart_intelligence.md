## 2. The Heart's Nervous System

### 2.1 The "Little Brain"

**Intrinsic Cardiac Ganglia:**
- ~40,000 neurons
- Called "heart brain" or "intrinsic cardiac nervous system"
- Can function independently
- Learns and remembers

### 2.2 Communication

```
HEART-BRAIN COMMUNICATION:
══════════════════════════

HEART → BRAIN (Afferent signals):
• More signals upward than downward!
• Via vagus nerve
• Via spinal cord
• Via hormones
• Via pressure waves

BRAIN → HEART (Efferent signals):
• Less than heart sends up
• Modulates but doesn't control
• Heart can override
```

### 2.3 Key Finding

**The heart sends MORE signals to the brain than vice versa.**

This reverses the assumed hierarchy.
