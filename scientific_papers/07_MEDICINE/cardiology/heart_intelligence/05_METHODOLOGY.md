## 5.3 CORRELATION METHODOLOGY

### Mapping Process

```
STEP 1: Identify frontend finding
        (Scientific discovery)
                |
                v
STEP 2: Search backend documentation
        (Traditional description)
                |
                v
STEP 3: Compare concepts
        (Structural matching)
                |
                v
STEP 4: Validate correlation
        (Evidence assessment)
                |
                v
STEP 5: Calculate confidence
        (Weighted scoring)
```

### Correlation Criteria

| Criterion | Definition | Weight |
|-----------|------------|--------|
| Concept match | Same phenomenon described | 30% |
| Mechanism fit | Explanation matches | 25% |
| Prediction fulfilled | Backend predicted frontend | 25% |
| Cross-validation | Multiple sources agree | 20% |

## 5.4 VALIDATION METRICS

### For Each Finding

```
EVIDENCE QUALITY:
=================

A (95-100%): Multiple peer-reviewed studies, replicated
B (85-94%): Peer-reviewed, some replication
C (75-84%): Limited peer-review, case studies
D (65-74%): Preliminary research, anecdotal
F (<65%): Insufficient evidence
```

### Confidence Calculation

```
C = (S_peer × 0.30) + (S_replication × 0.25) + 
    (S_mechanism × 0.25) + (S_backend × 0.20)

WHERE:
S_peer = Peer-review score (0-100)
S_replication = Replication score (0-100)
S_mechanism = Mechanism clarity (0-100)
S_backend = Backend correlation (0-100)
```

## 5.7 CONTROLS FOR BIAS

### Potential Confounds

| Confound | Control |
|----------|---------|
| Confirmation bias | Use null hypothesis framework |
| Selection bias | Include negative studies |
| Cultural bias | Cross-cultural validation |
| Measurement bias | Multiple measurement methods |
