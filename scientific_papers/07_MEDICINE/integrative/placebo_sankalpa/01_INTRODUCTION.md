## 1.1 THE PLACEBO MYSTERY

The placebo effect is one of medicine's most reliable yet least understood phenomena. Sugar pills relieve pain. Sham surgeries heal knees. Fake acupuncture works.

### The Core Question

```
THE FUNDAMENTAL PUZZLE:
=======================

HOW can an inert substance produce REAL physiological changes?

• Sugar pill -> Pain relief (measurable)
• Saline injection -> Immune changes (blood tests)
• Sham surgery -> Joint healing (follow-up confirmed)

This is not "just psychological" —
it produces OBJECTIVE biological changes.
```
