## 5.4 NOCEBO INVESTIGATION

### Evidence Categories

1. **Side effect studies:**
   - Warnings increase side effects
   - Same drug, different label -> different effects

2. **Negative prognosis:**
   - "Terminal" diagnosis effects
   - Hopelessness outcomes

3. **Cultural beliefs:**
   - Voodoo death documentation
   - Cross-cultural negative beliefs

### Symmetry Validation

- Same mechanism as placebo (reversed)
- Same brain areas (opposite activation)
- Same neurochemistry (opposite direction)
