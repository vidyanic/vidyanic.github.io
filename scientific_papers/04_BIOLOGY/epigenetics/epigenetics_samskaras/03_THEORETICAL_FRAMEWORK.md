## 3.3 The Inheritance Problem

### 3.3.1 How Marks Transmit

```
PARENT GENERATION:
Experience -> Epigenetic mark -> Some marks in gametes

OFFSPRING GENERATION:
Inherited marks -> Expressed patterns -> Behavior/health

MECHANISM:
Not all marks transmit (most erased)
But some escape reprogramming
Particularly stress-related marks
```

### 3.3.2 Karma Parallel

```
BACKEND MODEL:
Actions/Experiences -> Samskaras formed -> Stored in Karmashaya

NEXT LIFE:
Karmashaya -> Prarabdha (active karma) -> Expression in life

MECHANISM:
Not all Samskaras activate (Sanchita remains)
But active portion shapes life (Prarabdha)
Particularly strong impressions
```

---

## 3.4 The Modification Problem

### 3.4.1 Epigenetic Plasticity

Marks can be:
- Added by experience
- Removed by opposite experience
- Modified by environment

### 3.4.2 Samskara Plasticity

Samskaras can be:
- Added by experience (new impressions)
- Purified by practice (Samskara Shuddhi)
- Modified by Sadhana
