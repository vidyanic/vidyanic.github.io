## 8.1 Plant as Sleep Mode

### 8.1.1 The Analogy

```
+===========================================================================+
|                    PLANT CONSCIOUSNESS = DEEP SLEEP                       |
+===========================================================================+
|                                                                           |
|   HUMAN STATES:                                                           |
|   ============                                                            |
|   Waking (Jagrat)       = Full awareness, active                         |
|   Dreaming (Swapna)     = Inner experience, passive                      |
|   Deep Sleep (Sushupti) = No thoughts, but consciousness present         |
|   Turiya               = Pure awareness                                  |
|                                                                           |
|   PLANT CONSCIOUSNESS ≈ SUSHUPTI (Deep Sleep)                            |
|   • Present but not reflective                                           |
|   • Experiencing without knowing                                         |
|   • Being without thinking                                               |
|   • Conscious but not aware of being conscious                           |
|                                                                           |
|   You were conscious in deep sleep.                                      |
|   You just don't remember.                                               |
|   Plants live in that state.                                             |
|                                                                           |
+===========================================================================+
```

### 8.1.2 The Evidence

When you wake from deep sleep:
- You feel refreshed (something happened)
- You say "I slept well" (experienced quality)
- But no memory of content

Plants live in this state continuously.
