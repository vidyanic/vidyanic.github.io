- **Svadharma (‡§∏‡•ç‡§µ‡§ß‡§∞‡•ç‡§Æ/self-purpose):** Heart cells pump, liver cells filter, etc.

### "Why does it replicate infinitely?"

Because that's the **default programming** for cells.

Normal cells have differentiated instructions: "Be a skin cell. Divide X times. Then stop."

Cancer cells lost the differentiation. They're running stem-cell code without limits ‚Äî like an NPC stuck in spawn mode.

Yes ‚Äî **differentiation therapy** (making cancer cells "remember" their identity) shows promising results in some cancers.
