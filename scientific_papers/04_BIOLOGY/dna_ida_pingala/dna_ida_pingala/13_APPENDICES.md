## APPENDIX D: EPIGENETIC MECHANISMS

### D.1 DNA Methylation

```
MECHANISM:
==========

CH₃ (methyl group) added to cytosine
at CpG dinucleotides

RESULT:
• Gene silencing
• Chromatin condensation
• Heritable modification

CORRELATION:
• Trauma -> Increased methylation at stress genes
• Meditation -> Decreased methylation at inflammation genes
• Aging -> Accumulated methylation patterns
```

### D.2 Histone Modification

```
MECHANISM:
==========

Histone proteins modified by:
• Acetylation (activation)
• Methylation (variable)
• Phosphorylation (signaling)
• Ubiquitination (degradation)

RESULT:
• Changes in DNA accessibility
• Gene expression altered
• Heritable in some cases
```

### D.3 Transgenerational Transmission

```
PATHWAY:
=======

Parent experience 
    -> Epigenetic marks in gametes
    -> Transmitted to offspring
    -> Expression in F1 generation
    -> Sometimes persists to F2, F3

EXAMPLES:
• Holocaust -> FKBP5 methylation in children
• Dutch Famine -> Metabolic changes in grandchildren
• Fear conditioning -> Olfactory sensitivity in pups
```
