# 4. HYPOTHESIS

---

## 4.1 Primary Hypothesis

**H1: Memory operates on three interconnected levels — local (brain), extended (field), and universal (Akashic) — with the brain functioning as a transceiver that accesses memory fields rather than as the sole storage location.**

---

## 4.2 Sub-Hypotheses

### H1a: Local Memory (Brain)

The brain stores frequently-accessed and working memory as a "cache" for rapid access, but this is not the complete memory store.

**Predictions:**
- Brain damage affects access speed, not total memory
- Memory can be restored if access is restored
- Local storage is limited and prioritized

### H1b: Extended Memory (Field)

Species and collective memories exist in extended fields accessible through intuition, instinct, and altered states.

**Predictions:**
- New behaviors can spread without direct learning
- Collective trauma affects subsequent generations
- Instincts exist without individual learning

### H1c: Universal Memory (Akashic)

All information ever generated is permanently stored in the universal Akashic field, accessible through specific consciousness states.

**Predictions:**
- Complete life reviews possible (NDE)
- Past-life memories can be accessed
- Skills can emerge without personal learning
- Memory persists beyond body death

---

## 4.3 Null Hypotheses

### H0a: Standard Model

Memory is solely stored in brain synaptic connections, and all memory phenomena can be explained by neural mechanisms.

**If true:**
- Brain damage = proportional memory loss (always)
- No access to non-personally experienced memories
- No memory during zero brain activity
- No terminal lucidity in damaged brains

### H0b: Selective Survival

Anomalies are rare exceptions, errors, or fraud, not genuine phenomena requiring a new model.

**If true:**
- Terminal lucidity cases are misreported
- Past-life memories are fantasy or leading
- Savant syndrome has neurological explanation
- Brain damage paradoxes have local explanations

---

---

## 4.5 Falsification Criteria

### H1 would be falsified if:

1. **All anomalies explained neurologically** — No need for extended model
2. **Terminal lucidity proven false** — All cases are misreporting
3. **Past-life cases all debunked** — No genuine verified cases
4. **No correlation with consciousness states** — Access doesn't improve
5. **Memory completely localized** — Found specific engram for all memories
