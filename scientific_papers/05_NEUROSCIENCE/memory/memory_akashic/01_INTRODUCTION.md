## 1.1 The Memory Enigma

Memory is central to human experience, yet its fundamental nature remains mysterious. Standard neuroscience assumes memory is stored in synaptic connections within the brain. However, multiple phenomena challenge this assumption:

- **Terminal lucidity:** Clear memory returns in patients with destroyed brain tissue
- **Past-life memories:** Children report detailed memories of previous lives
- **Savant syndrome:** Sudden access to complex skills without prior learning
- **Brain damage paradoxes:** Normal function with minimal brain tissue
- **NDE reports:** Complete life reviews during clinical death

These anomalies suggest that memory may operate beyond the physical brain.

1. Brain damage should cause proportional memory loss

3. Memories cannot exist outside the physical brain
4. No access to memories never personally experienced
