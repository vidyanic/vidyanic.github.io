## 8.7 WHY DOCTORS CAN'T "FIX" DEATH

### Level Mismatch Problem

```
MEDICAL PERSPECTIVE:
====================
Doctor examines dead person:
• Heart can be restarted (sometimes)
• Lungs can be ventilated
• Blood can be circulated
• But person doesn't "come back"

WHY?
• Doctor is treating Level N (physical)
• Problem is at Level N-5 (Atman left)
• Cannot fix from wrong level!

ANALOGY:
• Game crashes (software issue)
• You repair the monitor (hardware)
• Game still doesn't work
• Because problem is not in monitor!
```

### The Real Sequence

```
WRONG: Heart stops -> Death
RIGHT: Atman leaves -> Prana withdraws -> Heart stops -> Death observed

TIMING:
• Atman decides to leave (karma exhausted)
• Prana begins withdrawal (hours before)
• Organs fail as consequence (minutes)
• Doctor declares death (too late to "fix")

This is why:
• CPR works sometimes (Atman hasn't fully left)
• CPR fails other times (Atman already gone)
• Technology cannot prevent death (wrong level)
```
