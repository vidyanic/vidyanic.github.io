## 6.1 NEAR-DEATH EXPERIENCE EVIDENCE

### 6.1.1 AWARE Study Findings (Parnia et al., 2014)

```
STUDY DESIGN:
- 2,060 cardiac arrest patients
- 330 survivors interviewed
- 140 reported memories/awareness
- 39 reported detailed NDE

KEY FINDING:
- Patient accurately described events
- Occurred during 3+ minutes of zero brain activity
- Vision, hearing not possible (eyes closed, unconscious)

IMPLICATION:
Consciousness continued after clinical death
-> Supports H₄ (NDE during separation)
```

### 6.1.2 van Lommel Study (2001)

```
STUDY DESIGN:
- 344 cardiac arrest patients
- 62 (18%) reported NDE
- 12% had deep NDE with all elements

KEY FINDINGS:
- Duration of arrest did NOT predict NDE occurrence
- Age, religion, prior knowledge did NOT predict NDE
- NDEs occurred during verified flat EEG

IMPLICATION:
NDE not produced by dying brain
-> Consciousness independent of brain activity
```

## 6.2 PRE-DEATH BEHAVIORAL CHANGES

### 6.2.1 Terminal Lucidity

```
PHENOMENON:
- Mental clarity in dying patients with dementia/delirium
- Occurs hours to days before death
- Person becomes "present" again unexpectedly

INCIDENCE:
- 5-10% of dementia patients
- Higher in hospice settings

INTERPRETATION (Multi-Level Model):
- Atman beginning withdrawal
- Temporary clarity as consciousness shifts levels
- "Last light" before internal sun dims
```

### 6.2.2 Pre-Death Dreams and Visions

```
COMMON ELEMENTS (Kerr et al., 2014):
- 88% of dying patients reported visions
- 72% saw deceased relatives
- 59% felt visited by deceased
- Dreams became more vivid approaching death

TIMING:
- Increased frequency in final 3-7 days
- Peaked at 24-48 hours before death

INTERPRETATION:
- Consciousness shifting between levels
- "Seeing" subtle realm as Atman withdraws
```

## 6.3 CPR OUTCOME ANALYSIS

### 6.3.1 Unexplained Patterns

```
PARADOX 1: Immediate CPR fails
- Some patients: CPR started within 1 minute
- Outcome: No recovery
- Standard explanation: Unknown

PARADOX 2: Delayed CPR succeeds
- Some patients: CPR delayed 5+ minutes
- Outcome: Full recovery, no brain damage
- Standard explanation: Unknown

MODEL EXPLANATION:
- Success depends on consciousness state
- If Atman fully withdrawn -> CPR fails
- If Atman still connected -> CPR can succeed
```

### 6.3.2 NDE "Choice" Reports

```
COMMON NDE ELEMENT:
"I had to decide whether to go back"
"Someone told me it wasn't my time"
"I chose to return for my children"

INTERPRETATION:
- Consciousness at decision point
- Return possible if Atman not fully departed
- Once fully departed, no return possible
```
