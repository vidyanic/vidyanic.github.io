## 5.3 Analysis Methods

```

━━━━━━━

1. List dark matter properties (cosmological)
2. List Loka/Guna properties (Backend)
3. Map structural correspondences
4. Identify mismatches
5. Evaluate explanatory power

EXAMPLE:
Dark matter property: Gravitates but doesn't emit light
Loka property: Other Lokas' Tamas affects spacetime but is invisible

```

4. Evaluate fit

Observed visible: 5%
