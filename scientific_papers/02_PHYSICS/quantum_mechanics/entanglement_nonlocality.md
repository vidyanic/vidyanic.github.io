## 1. What Is Entanglement?

### 1.1 Basic Description

When particles become entangled:
- Their properties become correlated
- Measuring one INSTANTLY determines the other
- Distance is irrelevant
- No signal passes between them

### 1.2 Example: Spin Entanglement

**Create two entangled photons:**
```
Before measurement:
Photon A: Spin = ^ AND v (superposition)
Photon B: Spin = ^ AND v (superposition)
Relationship: ALWAYS OPPOSITE

After measuring A (result: ^):

Photon B: Spin = v (INSTANTLY determined)
```

## 3. Bell's Theorem (1964)

### 3.1 The Test

**John Bell created a mathematical test:**
- If hidden variables exist -> Certain statistical limit (Bell inequality)
- If quantum mechanics is correct -> Inequality violated

### 3.2 The Result

**Experiments (Aspect 1982, many since):**
```
Bell inequality: VIOLATED!

Meaning:
‚Ä¢ Hidden variables DON'T explain entanglement
‚Ä¢ Quantum mechanics is correct
‚Ä¢ Non-locality is REAL
```

### 3.3 Nobel Prize 2022

**Alain Aspect, John Clauser, Anton Zeilinger:**
- Proved beyond doubt that entanglement is real
- Closed all experimental loopholes
- Non-locality confirmed at highest level
