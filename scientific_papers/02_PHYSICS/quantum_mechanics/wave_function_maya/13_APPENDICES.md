## Appendix A: Key Quantum Formulas

```
Time-dependent:
iℏ ∂|ψ⟩/∂t = Ĥ|ψ⟩

Time-independent:
Ĥ|ψ⟩ = E|ψ⟩
```

### Born Rule

```
P(k) = |⟨φₖ|ψ⟩|² = |cₖ|²
```

### Heisenberg Uncertainty

```
ΔxΔp ≥ ℏ/2
ΔEΔt ≥ ℏ/2
```
