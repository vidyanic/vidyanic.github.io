# 8. Backend Analogy: The Guna System

## 8.1 The Guna Framework

### 8.1.1 Origin and Definition

The Guna system comes from Samkhya philosophy, one of the oldest systematic frameworks for understanding reality.

```
THE THREE GUNAS:
━━━━━━━━━━━━━━━━

SATTVA (सत्त्व) — "Being-ness"
• Derived from "sat" (truth, existence)
• Light, clarity, intelligence
• Tendency toward order
• Physical: Negentropy, information

RAJAS (रजस्) — "Dust/Activity"
• Derived from "raj" (to color, to agitate)
• Motion, passion, energy
• Tendency toward change
• Physical: Kinetic energy, temperature

TAMAS (तमस्) — "Darkness"
• Derived from "tam" (to perish, to darken)
• Inertia, dullness, dissolution
• Tendency toward equilibrium
• Physical: Entropy, mass
```

### 8.1.2 The Conservation Law

```
FUNDAMENTAL CONSTRAINT:
Sattva + Rajas + Tamas = 1

PROPERTIES:
• Always normalized to unity
• Transformation, not creation
• One increases -> Others decrease
• Energy-like conservation
```
