## Appendix A: Mathematical Details

### A.1 Boltzmann's Derivation

```
STATISTICAL DEFINITION:
S = k·ln(W)

DERIVATION:
1. System has many microstates (W)
2. All microstates equally probable
3. Entropy measures "spread" over microstates
4. Logarithm ensures additivity: S_total = S_1 + S_2

PROPERTIES:
• S ≥ 0 (always non-negative)
• S = 0 only for W = 1 (single state)
• S increases with W
```

### A.2 Clausius Inequality

```
∮ δQ/T ≤ 0

FOR IRREVERSIBLE PROCESSES:
∮ δQ/T < 0

FOR REVERSIBLE:
∮ δQ/T = 0

IMPLICATION:
Entropy is a state function
Irreversible processes increase universe entropy
```

### A.3 Guna Normalization

```
CONSTRAINT:
S + R + T = 1

DIFFERENTIAL FORM:
dS + dR + dT = 0

IMPLICATION:
Any increase in one Guna requires decrease in others
Conservation law
```
