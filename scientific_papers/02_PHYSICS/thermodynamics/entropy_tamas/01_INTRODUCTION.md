# 1. Introduction

## 1.1 The Entropy Problem

The Second Law of Thermodynamics states that in any isolated system, entropy never decreases:

```
dS/dt ≥ 0

WHERE:
S = Entropy (measure of disorder)
t = Time
```

This simple law has profound implications:
- Time has a direction (the "arrow of time")
- Order always degrades to disorder
- Heat flows from hot to cold, never reverse
- The universe is heading toward "heat death"

**The problem:** Why? Why does disorder always win?

---

## 1.2 What Is Entropy?

### 1.2.1 Thermodynamic Definition

```
dS = δQ_rev / T

WHERE:
dS = Change in entropy
δQ_rev = Heat transferred reversibly
T = Temperature
```

### 1.2.2 Statistical Definition (Boltzmann)

```
S = k·ln(W)

WHERE:
S = Entropy
k = Boltzmann constant (1.38×10⁻²³ J/K)
W = Number of microstates

MEANING:
More microstates = More entropy = More disorder
```

### 1.2.3 Information Definition (Shannon)

```
H = -Σ p(i)·log₂(p(i))

WHERE:
H = Information entropy
p(i) = Probability of state i

MEANING:
More uncertainty = More entropy = Less information
```
