## 8.1 THE SAVE FILE MODEL

### Player-Avatar Relationship

```
REINCARNATION = NEW GAME+
===========================

PLAYER (Atman):
• The conscious entity
• Persists across games
• Carries progress

SAVE FILE (Subtle Body):
• Samskaras = Skill levels
• Vasanas = Preferences
• Karma = Quests incomplete

AVATAR (Physical Body):
• New character each game
• Fresh appearance
• New environment

NEW GAME+:
• Start new game
• With accumulated stats
• New challenges based on level
```

## 8.2 THE DATA TRANSFER MODEL

### What Transfers

```
PLAYER PROFILE (Atman + Subtle Body):
|
+-- PERMANENT:
|   +-- Consciousness (the player)
|   +-- Core tendencies (preferences)
|   +-- Accumulated karma (quest log)
|
+-- SEMI-PERMANENT:
|   +-- Skills (with decay)
|   +-- Emotional patterns
|   +-- Relationship marks
|
+-- CLEARED:
    +-- Physical memory (brain-based)
    +-- Body appearance
    +-- Environment location

NEW BODY = New hardware running same user
```

### Memory Leak Analogy

```
WHY CHILDREN REMEMBER:
=====================

1. NEW HARDWARE (Brain)
   • Fresh installation
   • Hasn't fully initialized
   • Old user data still accessible

2. BY AGE 7-8
   • System fully configured
   • "Clean install" complete
   • Old data overwritten

3. TRAUMATIC MEMORIES
   • Marked as "critical"
   • Harder to overwrite
   • Persist longer

This is a "memory leak" from previous user session.
```

## 8.4 BIRTHMARK AS DATA CORRUPTION

### The Physical Marker Mechanism

```
WOUND -> IMPRINT -> BIRTHMARK:
============================

1. TRAUMATIC DEATH (e.g., bullet wound)
   • Creates intense samskara
   • Marks subtle body template
   • Location encoded

2. AT REBIRTH
   • Subtle body provides template
   • Physical development follows template
   • Damaged area -> developmental deviation

3. RESULT
   • Birthmark at corresponding location
   • Shape may match wound
   • Like corrupted file affecting new installation

ANALOGY:
Like damaged installer affecting new software
"Installation error at address X" = Birthmark at location X
```
