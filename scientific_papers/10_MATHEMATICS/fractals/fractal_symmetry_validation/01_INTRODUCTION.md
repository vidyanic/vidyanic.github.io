## 1.4 What is a Fractal?

### 1.4.1 Mathematical Definition

A fractal is a geometric object that:
1. Has self-similarity at different scales
2. Has fractional dimension
3. Is generated by recursive processes

### 1.4.2 Examples in Nature

| Natural Fractal | Self-Similarity |
|-----------------|-----------------|
| Coastline | Same jaggedness at all zoom levels |
| Tree | Branches repeat trunk pattern |
| Blood vessels | Branching pattern repeats |
| Galaxy distribution | Clusters at all scales |
| Romanesco broccoli | Spiral repeats within spiral |

### 1.4.3 Backend Extension

The Backend claims nature is fractal not just in **geometry** but in:
- **Structure** (layers, axes)
- **Process** (cycles)
- **Information** (patterns)
- **Consciousness** (observer at each level)
