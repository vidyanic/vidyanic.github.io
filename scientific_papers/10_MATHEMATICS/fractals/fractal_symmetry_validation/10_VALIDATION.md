```

r_n ≈ 0.4 + 0.3 × 2^n (Titius-Bode)

GENERAL:

```

τ_atom ~ 10^-10 s < τ_cell ~ 10^6 s < τ_human ~ 10^9 s < τ_planet ~ 10^17 s

RATIO: τ(N+1) / τ(N) ~ 10^3 to 10^6
```

### 10.1.4 c Preservation

```
c = λ / τ = CONSTANT at all levels

WHERE:
λ = characteristic length at level N
τ = characteristic time at level N
