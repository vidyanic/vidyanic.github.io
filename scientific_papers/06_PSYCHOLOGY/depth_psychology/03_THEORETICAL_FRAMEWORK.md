## 3.1 The Antahkarana Model

### 3.1.1 Definition

**Antahkarana (अन्तःकरण)** = "Inner Instrument"

The complete inner apparatus through which consciousness interfaces with the world.

### 3.1.2 Four Components

```
+==============================================================+
|                     ANTAHKARANA                               |
|                   (Inner Instrument)                          |
+==============================================================+
|                                                              |
|   1. CHITTA (चित्त) — Mind-Stuff / Memory                   |
|      +-- Stores all Samskaras and Vasanas                    |
|      +-- The "unconscious" storehouse                        |
|      +-- Contains potential for all mental activity          |
|                                                              |
|   2. MANAS (मनस्) — Processing Mind                         |
|      +-- Coordinates senses                                  |
|      +-- Produces thoughts (Vikalpa)                         |
|      +-- Doubts, questions, processes                        |
|                                                              |
|   3. BUDDHI (बुद्धि) — Intellect / Discriminator            |
|      +-- Decides, judges                                     |
|      +-- Distinguishes real from unreal                      |
|      +-- Higher cognitive function                           |
|                                                              |
|   4. AHAMKARA (अहंकार) — Ego / I-Maker                      |
|      +-- Creates sense of "I"                                |
|      +-- Identifies consciousness with body-mind             |
|      +-- The root of individual separation                   |
|                                                              |
+==============================================================+
```

## 3.2 Chitta in Detail

### 3.2.1 Function

**Chitta serves as:**
1. **Storage** — All experiences leave impressions (Samskaras)
2. **Tendency** — Stored impressions create latent tendencies (Vasanas)
3. **Substrate** — The "screen" on which all mental activity appears
4. **Continuity** — Persists across death, carries Samskaras to next life

### 3.2.2 Samskaras (संस्कार)

**Definition:** Impressions left by every experience

```
EXPERIENCE -> SAMSKARA -> STORED IN CHITTA -> ACTIVATES LATER
     |                                           |
     +-- Every action, thought, perception ------+
```

**Types:**
- **Punya Samskaras** — Positive impressions (Sattvic)
- **Papa Samskaras** — Negative impressions (Tamasic/Rajasic)
- **Mixed** — Complex experiential residue

### 3.2.3 Vasanas (वासना)

**Definition:** Latent tendencies arising from Samskaras

**Mechanism:**
1. Samskaras accumulate
2. Similar Samskaras cluster
3. Clusters become Vasanas (tendencies)
4. Vasanas drive future action
5. Action creates more Samskaras -> cycle continues

### 3.2.4 Chitta and Consciousness

**Critical distinction:**
- Chitta is NOT consciousness
- Chitta is MATTER (Prakriti) in subtle form
- Atman (consciousness) illuminates Chitta
- The goal is to still Chitta, revealing Atman

## 3.3 Mapping to Depth Psychology

### 3.3.1 Freud's Topographical Model

| Freudian | Chitta Model |
|----------|--------------|
| Conscious | Surface Chitta + Manas activity |
| Preconscious | Accessible Samskaras |
| Unconscious | Deep Chitta, buried Samskaras |

### 3.3.2 Freud's Structural Model

| Freudian | Antahkarana |
|----------|-------------|
| Id | Tamas + Rajas-driven Vasanas |
| Ego | Ahamkara + Manas |
| Superego | Buddhi (discriminative function) + Dharmic Samskaras |

### 3.3.3 Jung's Model

| Jungian | Vedic |
|---------|-------|
| Personal Unconscious | Individual Chitta |
| Collective Unconscious | Hiranyagarbha (Cosmic Mind) |
| Archetypes | Devas (universal cosmic patterns) |
| Complexes | Vasana clusters |
| Shadow | Tamas accumulation in Chitta |
| Anima/Animus | Shakti/Shiva principles |
| Self | Atman (approached but not realized) |
| Individuation | Yoga path (incomplete) |

## 3.4 The Cosmic Dimension

### 3.4.1 Hiranyagarbha

**Definition:** The "Golden Womb" — cosmic mind of the universe

**Relationship to Chitta:**
- Individual Chitta is a portion of Hiranyagarbha
- Like waves in ocean, individual minds in cosmic mind
- Explains archetypal universality
- Source of collective unconscious

### 3.4.2 Devas as Archetypes

**Devas (देव)** = Cosmic principles/forces

| Archetype (Jung) | Deva (Vedic) | Function |
|------------------|--------------|----------|
| Great Mother | Shakti/Devi | Creative, nurturing, destructive |
| Father | Vishnu/Brahma | Preservation, creation |
| Hero | Indra/Rama | Conquest, courage |
| Trickster | Narada | Disruption, wisdom through chaos |
| Wise Old Man | Shiva/Guru | Transcendent wisdom |
| Child | Kumara/Ganesha | Beginnings, innocence |
| Self | Ishvara | Wholeness beyond ego |

### 3.4.3 Why Archetypes Are Universal

**Vedic explanation:**
1. All individual Chittas arise from Hiranyagarbha
2. Devas are the structural patterns of cosmic mind
3. Every individual Chitta inherits these patterns
4. Therefore, archetypes appear cross-culturally

## 3.5 Beyond Depth

### 3.5.1 Where Western Depth Stops

**Goal of depth psychology:**
- Ego integration
- Healthy adaptation
- Functional personality
- Meaning-making

### 3.5.2 Where Vedic Continues

**Goal of Yoga:**
- Chitta-vritti-nirodhah (stilling fluctuations)
- Identification with Atman, not Chitta
- Transcendence of ego (not integration)
- Moksha (liberation from cycles)

## 3.6 Theoretical Implications

### 3.6.1 For Psychology

1. **The unconscious is real** — Both traditions confirm
2. **Deeper than Freud** — Collective/cosmic dimension exists
3. **Archetypes are ontological** — Not just psychological
4. **Complete map exists** — In Vedic tradition

### 3.6.2 For Therapy

1. **Western methods valid** — Address personal Chitta
2. **Vedic methods add depth** — Address Vasana roots
3. **Integration possible** — Combine approaches
4. **Higher goal available** — Beyond adjustment to liberation

### 3.6.3 For Human Development

1. **Ego work necessary** — Stage on the path
2. **But not sufficient** — Only halfway
3. **Trans-ego possible** — Yoga leads further
4. **Liberation real** — Moksha is achievable
